  playableOffline

Returns YES if this instance refers to an offline video playable from the device’s local storage. Returns NO if the video has not completed downloading, or if the video has been purged and needs to be re-downloaded. This happens automatically if iOS needs to reclaim storage space for proper functioning of the device.
